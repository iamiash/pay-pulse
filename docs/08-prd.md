# Product Requirement Document (PRD) - PayPulse

1. Introduction & Objectives
PayPulse is a lightweight, web-based subscription tracker built specifically for students managing trial periods and recurring expenses. The goal is to provide a zero-friction dashboard where users can log, track, and get reminders for their digital expenses.

2. Core Features (Minimum Viable Product - MVP)
Subscription Dashboard: A clean list view showing all tracked items with fields for Service Name, Cost, Billing Cycle (Monthly/Yearly), and Next Renewal Date.
Trial Expiration Indicators: Automated visual badges (Green = Safe, Yellow = Expiring within 3 days, Red = Expiring in less than 24 hours).
Total Expense Aggregator: A dynamic counter calculating the user's total tech expenditure per month, converted to local currency.
Local Data Storage: No user account creation required; data is securely saved in the browser's local storage for absolute privacy.

3. Future Scope (Out of Scope for This Semester)
Automated email or SMS alert integrations.
Direct linking with real bank credit card accounts.
Shared family-plan bill splitting utilities.
