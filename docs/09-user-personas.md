
To design the ideal user interface, I created a primary user persona based on common student patterns discovered during interviews.

#### Goals & Needs:
Wants to try new developer tools and premium software using student discount free trials.
Needs a visual way to see exactly when a trial ends so he can cancel it if it isn't useful.
Wants to keep his overall tech spending under $25/month.

#### Frustrations & Pain Points:
Forgets about trial expiration dates because school assignments take up all his attention.
Hates banking apps that make it hard to see upcoming pre-authorized recurring international payments.
Reluctant to use heavy budgeting software that demands real bank account syncs.
